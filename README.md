# Technologies Used
This project is using Next.js with serverless functions, Bootstrap, and MongoDB 

# Prerequisites
Following steps needed for the setup

## Step 1:

``git clone https://github.com/phondani0/howtotech-nextjs.git``

## Step 2:

``npm install``

## Step 3:

``npm start``
